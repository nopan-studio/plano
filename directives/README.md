# Directives Directory

This directory contains Standard Operating Procedures (SOPs) for the AI agent to follow.

## Structure
Each directive should follow a standard format:
- **Goal**: What are we trying to achieve?
- **Inputs**: What data does the script or tool need?
- **Tools/Scripts**: Which MCP tools or scripts should be used?
- **Outputs**: What is the expected result?
- **Edge Cases**: What should the agent watch out for?

## Directives
- **plano_project_oversight.md**: Manages project progress, tasks, and status reports for the Plano project.
