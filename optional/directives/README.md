# Directives Directory

This directory contains Standard Operating Procedures (SOPs) for the AI agent to follow. These directives provide deterministic instructions for the agent to use the Plano MCP tools effectively within this project.

## Structure
Each directive follows a standard format:
- **Goal**: What are we trying to achieve?
- **Keywords & Triggers**: When should this directive be activated?
- **Tooling Constraints**: Specific rules for using MCP tools or scripts.
- **Procedure**: Step-by-step instructions for the agent.
- **Edge Cases**: What should the agent watch out for?

## Standard Directives
- **PLANO_PROJECT_OVERSIGHT.md**: Manages project progress, tasks, and status reports for the project.
- **PLANO_POST_UPDATE_STANDARD.md**: Defines the detailed standard for progress updates via `mcp_plano_post_update`.
- **PLANO_QA_DIRECTIVE.md**: Systematic identifying, reporting, and verifying defects within any project.
